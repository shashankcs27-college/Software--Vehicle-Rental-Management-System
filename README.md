# Vehicle Rental Management System

## Description
A web-based application to manage vehicle rentals. Users can book vehicles and admins can manage vehicles and bookings.

## Features
- User login/signup
- Vehicle listing
- Booking system
- Admin panel

## Tech Stack
- HTML, CSS, JavaScript
- Python Flask
- SQLite

## How to Run
1. Install Python
2. Install Flask
3. Run app.py
4. Open localhost
